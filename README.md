# Automobile-Sales-Visualization-Dashboard
_Interactive dashboard for automobile sales statistics using Python, Plotly, and Dash. 
This project presents an interactive Automobile Sales Statistics Dashboard, meticulously crafted using Dash for the web application framework and Plotly for advanced data visualization. The dashboard empowers users to explore and analyze historical automobile sales data, offering insights into both general yearly trends and specific patterns observed during economic recession periods.
Key Features:_

__Dynamic Data Visualization__: Utilizes Plotly to generate various types of charts, including line plots, bar charts, and pie charts, to effectively represent complex sales data.
Interactive Filtering: Users can switch between 'Yearly Statistics' and 'Recession Period Statistics' to focus on different aspects of the data. For yearly statistics, a dropdown allows selection of specific years.

__Recession Analysis__: Dedicated views to analyze average automobile sales, average sales by vehicle type, advertising expenditure share, and the effect of unemployment rates on sales during recessions.
Yearly Trends: Provides visualizations for overall average automobile sales, total monthly sales, average sales by vehicle type, and total advertising expenditure by vehicle type for any selected year.
Tools & Libraries Used:

__Dash__: The primary Python framework for building analytical web applications.
__Plotly__: For creating interactive and publication-quality graphs.
__Pandas__: For efficient data manipulation and analysis.
__Python__: The core programming language.

### __Skills Demonstrated:__

__Data Loading & Preprocessing__: Handling CSV data and basic cleaning (e.g., converting data types, filtering).
__Data Analysis__: Grouping and aggregating data to derive meaningful statistics.
Interactive Dashboard Development: Designing and implementing user interfaces with Dash components (Dropdown, HTML Divs, dcc.Graph).
__Callback Mechanisms__: Implementing Dash callbacks to enable dynamic updates and interactivity between components.
__Data Visualization__: Creating various chart types to convey insights effectively.
